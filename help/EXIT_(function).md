## _EXIT (function)
---
<blockquote>

### The _EXIT function prevents the user from closing a program and indicates if a user has clicked the close button in the window title ( X button) or used CTRL + BREAK.

</blockquote>

#### SYNTAX

<blockquote>

`exitSignal% = _EXIT`

</blockquote>

<blockquote>
