## KEY LIST
---
<blockquote>

### The KEY n statement is used to assign a "soft key" string or a flag and scan code to a function key or display function soft key assignments.

</blockquote>

#### SYNTAX

<blockquote>

`KEY n% , textString$`

</blockquote>
