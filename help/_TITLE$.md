## _TITLE$
---

### The _TITLE$ function returns the program title last set by the _TITLE statement.

#### SYNTAX

`currentTitle$ = _TITLE$`

#### DESCRIPTION
* If no title has been set, the title bar will say "Untitled" and this function will return an empty string ("").
