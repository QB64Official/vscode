## _NUMLOCK (function)
---
<blockquote>

### The _NUMLOCK function returns the current state of the Num Lock key as on (-1) or off (0).

</blockquote>

#### SYNTAX

<blockquote>

`keyStatus%% = _NUMLOCK`
