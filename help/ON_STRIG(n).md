## ON STRIG(n)
---
<blockquote>

### The ON STRIG(n) statement is an event procedure that directs program flow upon the press of a specified joystick button.

</blockquote>

#### SYNTAX

<blockquote>

`ON STRIG ( buttonFunction ) GOSUB { lineNumber | lineLabel }`

</blockquote>
