# _SCREENSHOW

The _SCREENSHOW statement can be used to display the main program window in a section of code.

  

## Syntax

_SCREENSHOW
  

## Description

* Use _SCREENSHOW to display the main program window after it was hidden by [_SCREENHIDE](_SCREENHIDE.md).
