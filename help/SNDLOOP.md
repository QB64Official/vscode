### The _SNDLOOP statement is like _SNDPLAY but the sound is looped. Uses a handle from the _SNDOPEN function.

</blockquote>

#### SYNTAX

<blockquote>

`_SNDLOOP handle&`

</blockquote>

#### DESCRIPTION

<blockquote>

*  Plays the sound identified by handle& in a loop.


</blockquote>
