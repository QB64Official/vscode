## _SCROLLLOCK (function)
---
<blockquote>

### The _SCROLLLOCK function returns the current state of the Scroll Lock key as on (-1) or off (0).

</blockquote>

#### SYNTAX

<blockquote>

`keyStatus%% = _SCROLLLOCK`
