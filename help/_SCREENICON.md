# _SCREENICON

The _SCREENICON statement can be used to minimize the main program window to the taskbar.

  

## Syntax

_SCREENICON
  

## Description

* Use _SCREENICON to minimize the main program window to the taskbar.

  

## Availability

* **QB64 v1.0 and up**
* **QB64-PE all versions**
