## _SCREENICON
---

### The _SCREENICON statement can be used to minimize the main program window to the taskbar.

#### SYNTAX

`_SCREENICON`

#### DESCRIPTION
* Use [_SCREENICON](./_SCREENICON.md) to minimize the main program window to the taskbar.
