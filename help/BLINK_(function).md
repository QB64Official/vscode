## _BLINK (function)
---
<blockquote>

### The _BLINK function returns the current blink setting for SCREEN 0 colors. If enabled, returns -1 (default), otherwise returns 0.

</blockquote>

#### SYNTAX

<blockquote>

`blinkState%% = _BLINK`

</blockquote>
