## EXIT FOR
---
<blockquote>

### The EXIT FOR statement allows a condition other than the counter value to exit a FOR...NEXT loop.

</blockquote>

#### SYNTAX

<blockquote>

`EXIT FOR`

</blockquote>

#### DESCRIPTION

<blockquote>

* Mostly used with a conditional statement to exit a [FOR](./FOR.md) counter loop early.
