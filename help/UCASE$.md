### The UCASE$ function outputs an all-uppercase version of a STRING .

</blockquote>

#### SYNTAX

<blockquote>

`result$ = UCASE$ ( text$ )`

</blockquote>

#### DESCRIPTION

<blockquote>

*  Used to guarantee that all alphabetical characters in a [STRING](STRING.md)  are capitalized.
*  Does not affect non-alphabetical characters.
