#### DESCRIPTION

<blockquote>

*  Specifies a variable's type in a declarative statement or parameter list using:
	* [DIM](DIM.md)  or [REDIM](REDIM.md) 
	* [DECLARE](DECLARE.md) [LIBRARY](LIBRARY.md) 
	* [SUB](SUB.md) 
	* [FUNCTION](FUNCTION.md)
