## EXIT DO
---
<blockquote>

### The EXIT DO statement can be used to exit a DO...LOOP from inside of the loop.

</blockquote>

#### SYNTAX

<blockquote>

`EXIT DO`

</blockquote>

#### DESCRIPTION

<blockquote>

* Used to exit a [DO](./DO.md) loop when a condition exists regardless of any loop conditions.


</blockquote>
