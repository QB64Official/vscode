### The SGN function returns the sign of a number value.

</blockquote>

#### SYNTAX

<blockquote>

`sign% = SGN(value)`

</blockquote>

#### EXAMPLES

<blockquote>



##### Example: Checking and changing negative values to positive ones.
```vb
n = -100
IF SGN(n) = -1 THEN n = ABS(n)
PRINT n
```
