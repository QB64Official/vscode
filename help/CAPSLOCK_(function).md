## _CAPSLOCK (function)
---
<blockquote>

### The _CAPSLOCK function returns the current state of the Caps Lock key as on (-1) or off (0).

</blockquote>

#### SYNTAX

<blockquote>

`keyStatus%% = _CAPSLOCK`
